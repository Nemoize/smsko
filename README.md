## Установка и запуск:

`apt-get install -y tor curl netcat`


```
git clone https://github.com/Nemoize/smsko
cd smsko
chmod +x smsko.sh
service tor start
./smsko.sh <НомерТелефона>
```

Параметры запуска
`-t - Время работы (def 120)`
`-T - Количество ТОР (def 5)`
